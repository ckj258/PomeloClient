一个简单的TCP长连接聊天demo.
客户端使用libpomelo2，基于coco2dx，服务器使用pomelo框.
兼容android mac win32多平台.
extend下附带相关库.
## Client

coco2dx引擎版本3.10，拷贝引擎文件至根目录编译，运行.

## Server

网易pomelo实例 

cd game-server .
npm install .
pomelo start .
